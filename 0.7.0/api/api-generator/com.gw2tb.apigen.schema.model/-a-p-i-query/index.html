APIQuery

data class APIQuery(source)

An APIQuery defines a type of request that can be made against the Guild Wars 2 API.

The path may differ from the endpoint's path as multiple paths may belong to a single logical endpoint in some cases.

The path may contain colon-prefixed (:) segments. These segments represent the path parameters. The real route can be constructed by substituting all such segments with an appropriate value for the respective parameter.

Since

0.7.0

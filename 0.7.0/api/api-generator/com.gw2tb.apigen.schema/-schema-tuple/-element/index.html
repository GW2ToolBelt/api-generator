data class Element(source)

An element of a tuple.

Type Coercion

An element of a primitive type may be marked as lenient. Such elements must be treated with special care as the element's type may not match the JSON type. Instead, the parser should coerce JSON strings into the property's type. If the string is empty, it should be treated as if the value is not set.

SchemaRecord

data class SchemaRecord : SchemaTypeDeclaration(source)

A SchemaRecord represents a JSON object.

Contrary to a SchemaMap, a record provides strong guarantees about the object's entries.

Mapping

SchemaRecords should be mapped to a representation that provides strong typing for all properties where possible (e.g. using a class in Java).
